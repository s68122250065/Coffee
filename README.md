# Coffee
Phase 1: คำถามอภิปราย
1.ทำไมลำดับการเสร็จไม่เหมือนเดิมทุกครั้ง?
ตอบ แต่ละ Thread ทำงานพร้อมกัน และระบบปฏิบัติการเป็นตัวจัดลำดับการทำงานของ Thread ทำให้บางครั้งเมนูหนึ่งอาจเสร็จก่อนหรือหลังต่างกันได้ ลำดับจึงไม่เหมือนเดิมทุกครั้งที่รันโปรแกรม
2.ถ้าเรียก run() แทน start() จะเกิดอะไรขึ้น?
ตอบ จะไม่เกิด Thread ใหม่โปรแกรมจะทำงานเหมือนเรียกเมธอดปกติทำให้ชงกาแฟทีละเมนูตามลำดับไม่ได้ทำพร้อมกัน
3.main thread ทำงานพร้อมกับ CoffeeMaker หรือไม่?
ตอบ ทำงานพร้อมกันเพราะเมื่อใช้ start() จะสร้างThreadใหม่ให้ CoffeeMaker ในขณะที่ main thread ยังทำงานต่อได้เลย เช่น พิมพ์ "รับออเดอร์ครบแล้ว" โดยไม่ต้องรอการชงกาแฟให้เสร็จก่อน
